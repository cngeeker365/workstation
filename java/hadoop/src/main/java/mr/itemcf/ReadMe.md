# 推荐算法

## 1. 分类

按数据使用划分：

*   协同过滤算法
    *   UserCF
    *   ItemCF
    *   ModelCF
*   基于内容的推荐
    *   用户内容属性
    *   物品内容属性
*   社会化过滤
    *   基于用户的社会网络关系
    
## 2. UserCF
> 通过用户对物品的评分来评测用户之间的相似性，基于用户之间的相似性做推荐。
> 即：给用户推荐和他兴趣相似的其他用户喜欢的物品。

## 3. ItemCF
> 通过用户对不同item的评分来评测item之间的相似性，基于item之间的相似性做推荐。
> 即：给用户推荐和他之前喜欢的物品相似的物品。
> * 注意：基于物品的协同过滤算法，是目前商用最广泛的推荐算法
